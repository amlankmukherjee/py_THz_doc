KeithleyClass module¶

class KeithleyClass.KeithleyClass(brd_idx: int, port_addr: int)[source]¶

Bases: GPIBBaseClass.GPIBBaseClass

Class to create Keithley objects connected by VISA-GPIB interface. The Keithley object is initiated in voltage mode by default.

Parameters

  • brd_idx – GPIB board index

  • port_addr – Port number for the communication port

DISPLAY_BOTTOM = 2¶

Chooses the left lower display of the connected Keithley

DISPLAY_TOP = 1¶

Chooses the main display of the connected Keithley

display_text(window, msg: str, wait_fcn)[source]¶

Function to display a message on the Keithley display.

Parameters

  • window – Area on which message is to be displayed. Use DISPLAY_TOP or DISPLAY_BOTTOM object attributes

  • msg – Message to be displayed on Keithley display. A maximum of 34 ASCII characters can be displayed

  • wait_fcn – Message is displayed until wait_fcn is being executed

Raises

InstrumentIOError – if set operation fails due to connection issues
