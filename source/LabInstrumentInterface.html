LabInstrumentInterface module¶

class LabInstrumentInterface.LabInstrumentInterface[source]¶

Bases: object

Interface for all lab instrument objects. Each type of “lab instrument” should inherit the LabInstrumentInterface. i.e., have defined functions of reading, writing and querying data. Read and query commands should return string values. Write and query commands should take commands/data as string for input
