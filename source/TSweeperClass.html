read_n_save(file_loc: str, file_name: str, read_wait_s: float, num_saves=1, save_timeout_s=60)[source]¶

Reads T-Sweeper scans continuously until either file limit or timeout is reached. Function saves each scan as a separate file in the specified file location.

Parameters

  • file_loc – Location for the files to be saved

  • file_name – Name of the file. An integer number is appended to the file name.

  • read_wait_s – Wait between data reads in seconds. Should be set equal to or slightly larger than the refresh rate of the TSweeper buffer. If set smaller than refresh rate, same data will be saved in the files until the buffer is updated.

  • num_saves – Number of files to be saved, default = 1

  • save_timeout_s – Timeout for the saving operation in seconds, default = 1 minute

Raises

InstrumentIOError – if read operations fails
