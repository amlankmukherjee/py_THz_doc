read_mag_pha_bulk(read_int: int, num_pts: int) → list[source]¶

Reads instrument data over a bulk time duration at specified time intervals. Maximum data points should be below 50,000 for single channel measurement and 25,000 for dual channel measurement

Parameters

  • read_int – Time interval in SECONDS for data logging

  • num_pts – Number of points to be read

Returns

2-D list of magnitude and phase (in degrees) data in format [MAG, PHA] and in [MAG1, PHA1, MAG2, PHA2] format for dual mode

Raises

  • InstrumentIOError – if operation fails due to connection issues

  • InstrumentConfigError – if data buffer size is exceeded
